## How It Works (Behind the Scenes)

When client clicks RUN.bat:

```
RUN.bat starts
  ↓
Check Python installed?
  ├─ Yes → Continue
  └─ No → Show error, client installs Python
  ↓
Install packages (pip install ...)
  ├─ vosk, numpy, scipy, scikit-learn, etc.
  └─ Silent, quick
  ↓
Check vosk_model folder exists?
  ├─ Yes → Skip download (client already has it)
  └─ No → Download model (1.4 GB, one time)
  ↓
Launch app.py
  ↓
Application starts!
```

## What IF There's an Error?

### "Python not found"
- Client installs Python from python.org
- Must check "Add to PATH"
- Restart computer
- Try RUN.bat again

### Download Error
- Check internet connection (yes, WiFi works!)
- Ensure 2GB free disk space
- Try again (RUN.bat auto-retries)

### Application crashes
- Close other programs
- Restart computer
- Try again
