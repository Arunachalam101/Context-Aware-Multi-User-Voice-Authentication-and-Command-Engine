## Troubleshooting

### RUN.bat doesn't work?
1. Make sure Python is installed
   - Download from: https://www.python.org/downloads/
   - During installation: CHECK "Add Python to PATH"
2. Restart your computer
3. Try RUN.bat again

### "No internet connection" error?
- Check WiFi/internet connection
- The model downloads only once (~1.4 GB)
- Wait 5-15 minutes, don't interrupt

### Application closes unexpectedly?
1. Close other programs (browsers, Discord, etc.)
2. Restart your computer
3. Try again

### Microphone not working?
- Check Windows Settings → Privacy → Microphone
- Enable microphone access
- Ensure microphone is not muted
